
# 🛍️ ShopHub

A modern e-commerce web application built with **React**, **React Router**, **React Context API**, and **React Hook Form**.

ShopHub demonstrates a complete frontend shopping experience including product browsing, authentication, cart management, checkout, and responsive UI.

---

## ✨ Features

- 🏠 Beautiful Home Page
- 🛒 Shopping Cart
- 📦 Product Details Page
- 🔐 User Authentication
- 👤 Login & Signup
- 💾 Local Storage Authentication
- ⚡ React Context API
- 🧾 Checkout Page
- ➕ Add to Cart
- ➖ Increase/Decrease Quantity
- ❌ Remove Products
- 🧹 Clear Cart
- 📱 Responsive Design

---

## 🖥️ Tech Stack

- React
- React Router DOM
- React Context API
- React Hook Form
- JavaScript (ES6+)
- CSS3
- HTML5
- Vite

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/DivyamTheDev/shophub.git
```

### Navigate to the project

```bash
cd shophub
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
